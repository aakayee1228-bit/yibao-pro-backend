## 🔴 审核风险评估

### 个人主体小程序限制

根据《微信小程序平台运营规范》，个人主体小程序**严禁**涉及：

1. ❌ **商业经营行为**
   - 报价、定价、交易
   - 合同、协议签订
   - 商业撮合服务

2. ❌ **涉及商业利益的功能**
   - 报价单生成（属于商业经营工具）
   - 商品定价管理
   - 客户报价记录

3. ✅ **允许的工具类**
   - 记账工具
   - 清单管理
   - 文档生成（不涉及商业利益）

## 🎯 解决方案

### 方案一：调整功能定位（推荐）

**核心思路**：将"报价单"改为中性的"清单/明细单"

#### 需要修改的内容

**1. 应用名称和定位**
```
❌ 易表单 - 报价单生成工具
✅ 易清单 - 商品清单管理工具
```

**2. 功能文案调整**
```
❌ 新建报价 → ✅ 新建清单
❌ 历史报价 → ✅ 历史清单
❌ 报价单 → ✅ 商品清单
❌ 报价单模板 → ✅ 清单模板
❌ 生成报价单 → ✅ 生成清单
```

**3. 页面路径**
```
❌ pages/quotes → ✅ pages/lists
❌ pages/quotes/create → ✅ pages/lists/create
```

**4. 数据库表名**
```
❌ quotes → ✅ lists
❌ quote_items → ✅ list_items
```

**5. 功能说明调整**
```
❌ "帮助商家快速生成专业报价单"
✅ "帮助用户快速生成商品清单和明细单"

❌ "提升成交效率"
✅ "提升工作效率"
```

**6. 类目选择**
```
✅ 工具 > 效率工具
✅ 工具 > 文档工具
```

#### 功能保留情况

| 功能 | 原名称 | 新名称 | 状态 |
|------|--------|--------|------|
| 商品管理 | 商品库 | 商品库 | ✅ 保留 |
| 客户管理 | 客户管理 | 客户管理 | ✅ 保留 |
| 单据生成 | 报价单 | 商品清单 | ✅ 改名 |
| 单据模板 | 报价单模板 | 清单模板 | ✅ 改名 |
| 数据统计 | 成交金额 | 记录统计 | ✅ 改名 |
