### 2. 功能说明

提交审核时说明：

```
本小程序是一款商品清单管理工具，主要功能：

1. 商品信息管理：帮助用户记录和管理商品信息（名称、规格、价格等）
2. 客户信息管理：帮助用户记录和管理客户联系方式
3. 清单生成：根据商品信息快速生成商品清单文档
4. 模板选择：提供多种清单格式模板

本应用为纯工具类应用，不涉及：
- 在线交易
- 支付功能
- 商业撮合
- 社交功能
- 广告展示

所有数据仅存储在云端数据库，保障用户数据安全。
```

### 3. 隐私协议要点

```
- 用户数据仅用于清单生成和管理
- 数据存储在云端数据库，保障安全
- 不收集用户隐私信息
- 不涉及支付、社交等敏感功能
- 用户可随时删除自己的数据
```
